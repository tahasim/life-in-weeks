# Productivity Apps

A collection of three powerful productivity and mindfulness apps designed to help you make better decisions and live intentionally.

## Apps Included

### 1. **Life in Weeks** (`index.html`)
Visualize your life in weeks. Enter your birth date to see how many weeks you've lived and how many remain. A powerful memento mori tool to inspire intentional living.

### 2. **Memento Mori: One Year Visualization** (`fear+death.html`)
Visualize one year of your life week by week. Meditate on mortality using the Stoic practice to gain clarity on what truly matters. Mark weeks you've lived and reflect on how you'll use the weeks remaining.

### 3. **Fear Setting Workshop** (`fearsetting.html`)
The classic fear-setting exercise to clarify your goals and overcome limiting beliefs. Structured prompts to help you define what you want and what's holding you back.

## Deployment

### Deploy to Vercel
1. Fork this repository or push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Connect your GitHub repository
4. Click "Deploy" - Vercel will auto-detect this is a static site

### Deploy to Netlify
1. Push this folder to GitHub (or any Git service)
2. Go to [netlify.com](https://netlify.com)
3. Click "New site from Git"
4. Connect your repository
5. Netlify will automatically handle the deployment

### Local Development
```bash
npm run dev
```

## No Dependencies
This is a pure HTML/CSS/JavaScript project with no build step required. All three apps work completely in the browser.
